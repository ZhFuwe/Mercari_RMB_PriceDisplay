# Mercari_RMB_PriceDisplay
适配 Mercari 主站、Shops 页、搜索页、详情页，显示换算的 RMB 价格，可修改汇率。


## 使用方式：
将脚本导入Tampermonkey、脚本猫等浏览器脚本管理插件并开启即可。

修改汇率后需刷新生效。
